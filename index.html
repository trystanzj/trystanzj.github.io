Farly is free, open source, offline, firewall policy management software.*


The Farly Automated Decision Support Framework is a firewall access-list configuration monitoring system.


The Farly Automated Decision Support Framework links your firewall configuration to the network with load balancer style active network probing.


Farly can be used to clean room reconfigure your firewalls.

*Farly supports Cisco ASA and FWSM firewalls.
